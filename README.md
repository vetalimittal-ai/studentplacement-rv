# 🎓 Student Placement Analytics Dashboard (Power BI)

---

## 📌 Project Overview
This project presents an *interactive Power BI dashboard* analyzing student placement performance across different branches, skills, and academic indicators.  
The dashboard provides insights into *CGPA, salary packages, DSA scores, projects, certifications, and internships*.

It helps stakeholders understand:
- Placement performance by branch  
- Impact of CGPA on salary  
- Role of certifications and aptitude  
- Student skill development (DSA & projects)  
- College tier distribution
